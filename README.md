# Vedic Gemstone AI Recommender 💎

A modern AI-assisted gemstone recommendation web app built with Next.js and Tailwind CSS. It suggests suitable gemstones based on user zodiac signs and lifestyle concerns using a structured recommendation logic.

## Features

* Zodiac & concern-based gemstone recommendations
* Dynamic result dashboard
* Sorting and filtering system
* Responsive SaaS-style UI
* Printable gemstone report
* Modular component architecture

## Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* Framer Motion

## Project Structure

```
app/
 ├─ page.tsx          # User input page
 └─ result/page.tsx   # Recommendation results

components/
 ├─ GemstoneCard.tsx
 ├─ MetricBanner.tsx
 └─ ResultHeader.tsx
```

## Run Locally

```bash
git clone https://github.com/Amit0022R/Gemstone-Recommendation-App.git

cd Gemstone-Recommendation-App

npm install

npm run dev
```

Open:

```
http://localhost:3000
```
